# Live2D / Cubism import guide

This folder is **path 2**: the same character, separated into layers that line up
on a shared canvas, ready to rig in **Live2D Cubism Editor** (mesh deformation,
hair/cloth physics, full head-turn and arm gestures). The web rig in the parent
folder is path 1.

## What's here

* `*.png` — each body part on its own **full-canvas** layer (so they overlay
  perfectly when stacked). Generated by `../build_rig.py`.
* `layers.json` — the canvas size and the recommended **stacking order** (`z`,
  back → front).

Canvas and order come straight from `layers.json`:

```
thighL, thighR  (z1)   →   shinL, shinR (z2)   →   torso (z3)
   →   foreL, foreR (z4)   →   head (z5)
```

## Import into Cubism (PSD route — recommended)

1. Stack the PNGs into a single **PSD**, one layer per file, ordered by the `z`
   in `layers.json` (thighs at the bottom, head on top). Any editor works:

   ```bash
   pip install pillow
   python3 - <<'PY'
   # quick PSD-less option: most artists just drag the PNGs into Photoshop/Krita
   # as layers in the z-order above and "Save As .psd".
   PY
   ```

   In Photoshop/Krita/GIMP: *File → New* at the `layers.json` canvas size, drag
   each PNG in as its own layer, name it after the file, order by `z`.
2. In **Cubism Editor**: *File → Import → PSD*. Each layer becomes an
   `ArtMesh`.
3. **Parent the deformers** to match the rig hierarchy:
   `hips → torso → {head, foreL, foreR}` and `hips → thighL/R → shinL/R`.
4. Add **rotation deformers** at the joints (neck, waist, cuffs, hips, knees) and
   key parameters: `ParamAngleX/Y/Z` (head look), `ParamBodyAngleX`, `ParamArmL/R`,
   plus a breath parameter on the torso.
5. Auto-generate meshes (*Modeling → Mesh → Auto*) and refine the head/hair for
   clean deformation.

## Notes

* The forearms are **skin-only** (the rolled sleeve cuff stays on the torso),
  which is also the cleaner split for Cubism. If you want a full shoulder raise,
  re-cut the arm to include the sleeve, or paint the upper arm on its own layer.
* The torso and thighs are already **inpainted** behind the resting arms, so the
  limbs can move without exposing holes.
* Colours/skin tone are the **original pixels** — nothing was recoloured.
